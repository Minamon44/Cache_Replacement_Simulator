# Cache Replacement Simulator

A visual tool built with Python to demonstrate different cache replacement policies including LRU, MRU, FIFO, LFU, and LIFO.

## Features
* **Visual Animation:** Watch how cache blocks change over time.
* **Multiple Policies:** Compare how different algorithms handle the same input sequence.
* **Interactive GUI:** Custom interface built with CustomTkinter.
