# Cache Replacement Simulator

A visual tool built with Python to demonstrate different cache replacement policies including LRU, MRU, FIFO, LFU, and LIFO.

## Features
* **Visual Animation:** Watch how cache blocks change over time through a dynamic timeline.
* **Multiple Policies:** Compare how different algorithms (LRU, MRU, FIFO, LFU, LIFO) handle the same input sequence.
* **Interactive GUI:** Modern interface built with CustomTkinter for easy parameter input.
